# 庁内DX推進ガイドライン（テスト版）

## 1. 目的
本ガイドラインは、当庁におけるデジタルトランスフォーメーション（DX）を効果的かつ安全に推進するための指針を定めるものである。

## 2. DX推進の基本ステップ
DXを推進するにあたっては、以下の4つのステップを順守すること。

### ステップ1: 現状分析 (As-Is)
- 各部署の業務フローを可視化し、アナログで行われている業務を洗い出す。
- 紙媒体の使用頻度、ハンコ決裁の有無などを数値化する。

### ステップ2: 目標設定 (To-Be)
- デジタル化によって達成すべき具体的な目標（例：処理時間の30%削減、ペーパーレス化率50%など）を設定する。
- 市民サービスの向上にどう繋がるかを明確にする。

### ステップ3: ツール定着化
- 導入したデジタルツール（チャットツール、Web会議、生成AIなど）の使用ルールを策定する。
- 職員向けの研修会を半期に1回実施する。

### ステップ4: 効果測定と改善
- 四半期ごとにKPIの達成状況を確認し、必要に応じて施策を見直す。

## 3. 生成AIの利用について
### 利用可能なAIモデル
- 当庁では、セキュリティ評価済みの「OWLight Chat」（Geminiモデル採用）のみを公式ツールとして認める。
- 個人アカウントでのChatGPT等の利用は、業務データを取り扱う場合に限り禁止とする。

### 入力データの制限
- **機密情報（マイナンバー、個人名、未公開の政策案など）は絶対に入力してはならない。**
- 入力するデータは、公開情報または庁内限りの一般情報（Level 2以下）に限る。

## 4. セキュリティインシデント対応
- 不審なメールを受信した場合や、ウイルス感染が疑われる場合は、直ちに**情報システム課（内線：9999）**へ連絡すること。
- 自己判断でPCを再起動したり、LANケーブルを抜いたりしないこと（ログ保全のため）。

## 5. テレワーク規定
- テレワーク実施時は、必ず庁内VPNを経由してシステムにアクセスすること。
- 公衆Wi-Fi（カフェや駅のフリーWi-Fi）の使用は原則禁止とする。やむを得ない場合は、支給されたモバイルルーターを使用すること。
